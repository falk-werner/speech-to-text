# Speech to Text

Example how to implement speech-to-text using [Vosk](https://alphacephei.com/vosk/).

## Usage

```bash
./speech-to-text.py
```

### Options

| Option      | Description |
|-------------|-------------|
| -d NUMBER   | select input device |
| -l LANGUAGE | choose language, default: de |
